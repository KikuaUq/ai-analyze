# QuantumLeap

Welcome to QuantumLeap, an AI-powered market intelligence platform providing unparalleled insights into US stocks and crypto markets.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Deployment**: [Vercel](https://vercel.com/)

---

## Deploying to Vercel (Recommended)

Follow these steps to deploy your application to Vercel and get a live, shareable URL.

### 1. Push to a Git Provider

Before deploying, your project needs to be in a GitHub, GitLab, or Bitbucket repository.

- [Create a new repository on GitHub](https://github.com/new).
- Follow the instructions to push your local project to the new repository.

### 2. Import Project to Vercel

1.  **Sign up or Log in**: Go to [vercel.com](https://vercel.com/) and create an account or log in. It's recommended to sign up with your Git provider account (e.g., GitHub).

2.  **Import Project**:
    - From your Vercel dashboard, click the "**Add New...**" button and select "**Project**".
    - Find your `QuantumLeap` repository and click the "**Import**" button next to it.

3.  **Configure and Deploy**:
    - Vercel will automatically detect that you are using Next.js and configure all the build settings for you.
    - You don't need to change any settings.
    - Click the "**Deploy**" button.

That's it! Vercel will now build and deploy your application. Once finished, you'll be given a live URL to view your site.
